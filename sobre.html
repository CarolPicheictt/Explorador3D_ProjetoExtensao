Sobre o Projeto

Olá! bem vindo! :)
Este projeto foi desenvolvido pelos acadêmicos Michel e Carolina da Universidade Estadual de Ponta Grossa (UEPG), como nota parcial do projeto de extensão 3, do segundo semestre do 2º ano do curso de Engenharia de Software.
O projeto é uma aplicação web baseada em HTML, CSS e JS, utilizando o framework A-frame. Seguindo com o objetivo de educação infantil de vocabulário em inglês.
O objetivo desse projeto é estimular e facilitar os alunos das escolas a aprender um novo idioma de forma divertida e descontraída, utilizando tecnologias simples!
